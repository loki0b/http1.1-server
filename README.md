## Tools
Manuais do Linux contém bastante informações relevantes para essa atividade, mas fique a vontade de pesquisar.

```man --whatis [keyword]``` Mostra quais manuais estão disponíveis para ***keyword*** que voce busca.
Exemplo:
``` bash
man --whatis socket
```

```man [manual page] [keyword]``` Mostra o manual ***page*** da ***keyword***.
Exemplo:
``` bash
man 2 socket
```
